# ГрінВуд — сайт крафтової кав'ярні

Односторінковий сайт кав'ярні «ГрінВуд»: головна сторінка, інтерактивне меню з кошиком та попереднім замовленням, програма лояльності з віртуальною карткою-штампом та карта точок самовивозу.

## Розділи сайту
- **Головна** — hero-блок, коротко про підхід кав'ярні, анонс програми лояльності
- **Меню** — 5 категорій (кава, альтернатива, чай, випічка, сніданки), фото, склад, об'єм і ціна кожної позиції
- **Кошик і передзамовлення** — вибір часу видачі замовлення та точки самовивозу
- **Лояльність** — картка з штампами «кожна 6-та кава в подарунок»
- **Контакти** — три точки в Дніпрі з графіком роботи та інтерактивною картою

## Технології
Чистий HTML, CSS та JavaScript в одному файлі — без збірки та залежностей. Шрифти підключені з Google Fonts, карта — вбудований OpenStreetMap.
